# image-BCP AGENTS

## 角色

- 本项目是图片基础卡片插件；正式职责是管理图片类基础卡片的查看、编辑、配置和资源生命周期。

## 开发规则

- 本地图片导入、资源 URL 解析、资源释放和资源删除必须通过正式宿主资源桥完成。
- 持久化配置只能写正式字段和值；本地资源路径统一写为相对于卡片根目录的路径，不得写回绝对路径、`blob:`、`data:` 或宿主临时 URL。
- 查看态、编辑态、schema、默认模板和 i18n 必须同步演进；不要出现界面行为、配置模型和文档三套口径。
- 视觉样式必须通过主题变量和正式样式接口点落地，不得为图片卡片私自硬编码整套视觉皮肤。
- 若图片插件需要新增公共资源协议、公共布局字段或生态级交互约定，先登记工单并同步共用文档。

## 验证

- 变更后至少运行与改动相关的 `lint`、`test`、`validate` 及对应单元/集成测试。
