# 富文本基础卡片插件重构开发计划（Milkdown 版）

版本：2026-03-25  
状态：已完成

---

## 1. 目标

围绕以下四个目标完成富文本基础卡片插件重构：

1. 把编辑内核切换到 Milkdown；
2. 用选区图标悬浮工具框与编辑区右键菜单替代旧版固定工具栏与操作窗口；
3. 建立 Markdown inline / file 双存储模型；
4. 重构查看态渲染链路与测试基线。

---

## 2. 阶段拆分

### Phase 1：契约与数据模型重构

- 收口新版 `RichTextCard` 配置契约
- 完成 schema、测试夹具与资源路径口径切换
- 清理旧 HTML 假设

### Phase 2：Milkdown 编辑器与图标悬浮工具框

- 接入 Milkdown 官方生态包
- 建立编辑器运行时与命令适配层
- 实现选区图标悬浮工具框、Tooltip、右键菜单与稳定输入链路

### Phase 3：Markdown 持久化与渲染重构

- 实现 200 字分流与根目录 Markdown 文件写入
- 收口 inline / file 双读取链路
- 重构查看态为静态 Markdown 渲染

### Phase 4：测试联调与验收

- 重写单元测试与集成测试
- 补齐关键交互验证
- 收尾 README、内部文档与验收清单

---

## 3. 阶段门禁

### Phase 1 完成门禁

- 新版配置模型定稿
- 旧 HTML 正文假设从 schema 层移除

### Phase 2 完成门禁

- 编辑器正式可编辑 Markdown
- 图标悬浮工具框、Tooltip 与右键菜单可稳定执行命令

### Phase 3 完成门禁

- 200 字阈值策略可稳定驱动 inline / file 切换
- 查看态可正确渲染两类内容来源

### Phase 4 完成门禁

- 目标测试全部通过
- 插件内部文档与 README 收口完成

---

## 4. 当前不在本计划内的扩展项

- 表格、公式、任务列表
- 图片/附件/媒体嵌入
- 协同编辑与评论
- 旧 HTML 配置兼容迁移

以上内容如需推进，需另行立项，不并入本次富文本基础卡片插件重构计划。
