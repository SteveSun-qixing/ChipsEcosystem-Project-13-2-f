# Phase 1 - 项目初始化与架构迁移（已完成）

## 目标

在不引入旧版架构耦合的前提下，基于新一代基础卡片脚手架创建富文本基础卡片插件工程，并完成最小可用的工程骨架。

## 任务列表

1. 初始化插件工程目录
   - 路径：`Chips-BaseCardPlugin/richtext-BCP`
   - 建立基础结构：`src/`、`config/`、`i18n/`、`templates/`、`tests/`
   - 状态：已完成

2. 配置基础元信息
   - `manifest.yaml` 中定义：
     - `id: chips.basecard.richtext`
     - `type: card`
     - `capabilities.cardTypes: [base.richtext]`
   - `package.json` 中定义脚本与 devDependencies（TypeScript、Vitest、jsdom）
   - 状态：已完成

3. 建立入口与运行时代码
   - `src/index.ts` 定义：
     - `BasecardRenderContext` / `BasecardEditorContext`
     - `renderBasecardView` / `renderBasecardEditor`
   - `src/render/runtime.ts` 与 `src/editor/runtime.ts` 提供挂载与卸载函数
   - 状态：已完成

4. 日志与配置
   - `config/logging.ts`：统一插件内部日志前缀为 `[basecard-richtext]`
   - `chips.config.mjs`：配置构建入口与输出目录
   - 状态：已完成

> 阶段结论：工程骨架已完全建立，可在后续阶段聚焦功能实现与重构。
