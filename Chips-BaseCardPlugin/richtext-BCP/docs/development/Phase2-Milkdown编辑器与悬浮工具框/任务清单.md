# Phase 2 - Milkdown 编辑器与图标悬浮工具框

状态：已完成

## 目标

用 Milkdown 取代旧版 `contenteditable` 编辑器，并建立选区触发的图标悬浮工具框与编辑区右键菜单交互。

## 任务列表

1. 接入 Milkdown 官方生态
   - 选择正式编辑器包与基础 preset
   - 建立编辑器实例创建、销毁与内容同步封装

2. 搭建命令适配层
   - 将按钮动作映射为 Milkdown 命令
   - 收口快捷键行为

3. 实现图标悬浮工具框
   - 监听选区变化
   - 基于选区矩形定位浮层
   - 处理点击命令后的焦点与选区恢复
   - 工具按钮收缩为图标热区并接入 Tooltip

4. 处理稳定输入问题
   - 组合输入（IME）
   - 粘贴归一
   - 编辑态滚动与浮层重定位

5. 实现编辑区右键菜单
   - 块级结构操作移入上下文菜单
   - 接入菜单键盘导航
   - 处理菜单打开与关闭时的浮层协同

## 验收标准

- 编辑器可加载并输出 Markdown
- 选中正文后图标悬浮工具框出现，取消选区后隐藏
- Tooltip 与右键菜单均符合生态契约
- 命令执行后不会丢失选区或打断输入法
