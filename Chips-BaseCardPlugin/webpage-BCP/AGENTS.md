# webpage-BCP AGENTS

## 角色

- 本项目是网页基础卡片插件；正式职责是提供网页卡片的查看态、编辑态、网页压缩包导入和配置模型。

## 开发规则

- 网页来源只能通过正式字段保存：远程 URL 或卡片根目录相对网页包路径。
- 网页压缩包导入、资源删除和资源 URL 解析必须走正式宿主链路，不得私接文件系统。
- 查看态、编辑态、schema、模板、i18n 与文档必须同步收口，不得形成多套口径。
- 若网页卡片需要新增公共 Bridge 契约、资源规则或文件格式约束，应同步更新生态共用技术文档。

## 验证

- 变更后至少运行与改动相关的 `lint`、`test`、`validate` 与单元/集成测试。
