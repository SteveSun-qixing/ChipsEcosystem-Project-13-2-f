# 书籍阅读器需求文档

## 目标

为薯片生态提供一个正式的电子书阅读器应用插件，聚焦“阅读器”本身，不引入书架、下载中心、设置中心等范围外模块。

## 本次正式范围

- 打开本地 EPUB / EPUB3 文件
- 被 Host 以 `resource.open` 或文件关联方式唤起
- 接受远程 EPUB 链接并在应用内读取
- 目录浏览
- 章节切换
- 章节内锚点跳转
- 封面页、正文页、图片页在统一阅读链路中正常渲染
- 封面页、整页插图、章节内独立插图在翻页 / 滚动两种模式下都必须响应式显示，不得溢出窗口，也不得出现明显异常留白
- 极简沉浸式阅读界面：
  - 空态只保留居中导入提示
  - 阅读态默认隐藏控制按钮
  - 点击阅读区中部后再显示导入、目录与阅读偏好操作
- 阅读模式切换：
  - 左右翻页
  - 上下滚动
- 阅读背景颜色切换
- 多输入设备阅读操作：
  - 鼠标
  - 键盘
  - 触摸板 / 滚轮
  - 触摸屏
- 基础阅读偏好：
  - 阅读模式
  - 背景颜色
  - 字号
  - 版心宽度
  - 衬线 / 无衬线字体
- 主题与多语言正式接入

## 不在本次范围

- 书架
- 用户级设置中心
- OPDS / 书源管理
- DRM / LCP
- 多格式全覆盖（如 PDF / MOBI / AZW）
- 复杂批注、书签、同步阅读进度

## 启动方式

- 应用启动器直接打开
- 通过系统文件选择器打开本地电子书
- 拖拽电子书文件到窗口
- Host 通过 `launchParams.targetPath`
- Host 通过 `launchParams.resourceOpen`
- 直接输入远程 EPUB 链接

## 正式约束

- Host 是唯一运行时承载
- 系统能力访问通过 `chips-sdk` / `window.chips.*`
- React 技术栈
- 文案全部接入 i18n
- 样式全部接入主题 token
- 使用正式 `manifest.yaml`
- 不引入书架、设置等无关功能代码
