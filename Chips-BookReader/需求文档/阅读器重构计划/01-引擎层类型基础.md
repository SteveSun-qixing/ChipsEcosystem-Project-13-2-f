# 阶段 01 — 引擎层类型基础

## 目标

定义渲染引擎层的公共类型系统，为后续所有引擎模块提供统一的类型契约。

## 背景

当前 `runtime.ts` 内部类型定义与具体实现交织在一起，缺乏独立的类型模块。重构后引擎层将拆分为分页引擎、滚动引擎、CSS 管理器、布局计算器、位置追踪器、动画控制器等多个模块，它们之间需要共享一套清晰的类型契约。

## 涉及文件

### 新建

- `src/engine/types.ts`

### 参考（只读）

- `src/domain/readium/runtime.ts` — 提取现有内部类型
- `src/domain/readium/layout.ts` — 布局相关类型
- `src/domain/epub/types.ts` — EpubThemePalette
- `src/utils/book-reader.ts` — ReaderPreferences, ReaderReadingMode

## 具体内容

### 需要定义的类型

```typescript
// ─── 方向与边界 ───
export type PageDirection = "previous" | "next";
export type ReadingBoundary = "start" | "end";
export type ReadingMode = "paginated" | "scroll";

// ─── 滚动行为 ───
export type ScrollBehavior = "auto" | "smooth";

// ─── 文档方向性 ───
export interface DocumentDirectionality {
  isVertical: boolean;          // 竖排书写模式
  isRtl: boolean;               // 右到左文本方向
}

// ─── 视口度量 ───
export interface ViewportMetrics {
  viewportWidth: number;         // iframe 可视宽度
  viewportHeight: number;        // iframe 可视高度
  scrollWidth: number;           // 内容总宽度
  scrollHeight: number;          // 内容总高度
  offsetWidth: number;           // 布局宽度（含边框）
  offsetHeight: number;          // 布局高度（含边框）
}

// ─── 分页度量 ───
export interface PaginationMetrics {
  totalColumns: number;          // CSS column 产生的总列数
  columnsPerSpread: number;      // 每个 spread 包含的列数（1 或 2）
  spreadCount: number;           // spread 总数
  currentSpreadIndex: number;    // 当前 spread 索引
  spreadWidthPx: number;        // 每个 spread 的像素宽度
  columnWidthPx: number;        // 每列的像素宽度
  columnGapPx: number;          // 列间距像素值
  totalScrollableWidth: number;  // 总可滚动宽度
  maxScrollOffset: number;       // 最大可滚动偏移量
}

// ─── 滚动度量 ───
export interface ScrollMetrics {
  scrollOffset: number;          // 当前滚动位置（纵向为 scrollTop）
  maxScrollOffset: number;       // 最大可滚动偏移
  viewportSize: number;          // 可视区域尺寸
  contentSize: number;           // 内容总尺寸
  scrollFraction: number;        // 滚动进度 0–1
}

// ─── 阅读位置锚点 ───
export interface ReadingPositionAnchor {
  spreadIndex: number;           // 当前 spread 索引
  spreadFraction: number;        // 在总 spread 中的比例 0–1
  scrollFraction: number;        // 通用滚动进度 0–1
}

// ─── 分页快照（用于窗口缩放恢复） ───
export interface PaginationSnapshot {
  offset: number;                // 当前像素偏移
  spreadWidth: number;           // 当前 spread 宽度
  maxOffset: number;             // 当前最大偏移
  spreadIndex: number;           // 当前 spread 索引
  totalSpreads: number;          // 总 spread 数
}

// ─── 阅读进度信息 ───
export interface ReadingProgress {
  sectionIndex: number;          // 当前章节索引
  sectionCount: number;          // 总章节数
  sectionTitle: string;          // 当前章节标题
  sectionFraction: number;       // 章节内进度 0–1
  bookFraction: number;          // 全书进度 0–1
  currentPage: number;           // 当前页码（章节内）
  totalPages: number;            // 总页数（章节内）
  displayLabel: string;          // 进度显示文案
}

// ─── 章节版式类型 ───
export type SectionKind = "chapter" | "illustration" | "full-page-image" | "cover";

// ─── 响应式布局结果 ───
export interface ResponsiveLayout {
  forcedColCount: "1" | "2";
  pageWidthPx: number;
  pageGutterPx: number;
  columnGapPx: number;
  maxLineLengthPx: number;
  illustrationWidthPx: number;
  shouldUseSpread: boolean;
}

// ─── 引擎事件 ───
export type EngineEventType =
  | "page-changed"
  | "scroll-changed"
  | "layout-changed"
  | "boundary-reached"
  | "progress-updated";

export interface EngineEvent {
  type: EngineEventType;
  timestamp: number;
}

export interface PageChangedEvent extends EngineEvent {
  type: "page-changed";
  direction: PageDirection;
  spreadIndex: number;
  totalSpreads: number;
}

export interface ScrollChangedEvent extends EngineEvent {
  type: "scroll-changed";
  scrollFraction: number;
}

export interface BoundaryReachedEvent extends EngineEvent {
  type: "boundary-reached";
  boundary: ReadingBoundary;
  direction: PageDirection;
}

export interface ProgressUpdatedEvent extends EngineEvent {
  type: "progress-updated";
  progress: ReadingProgress;
}

// ─── 主题色板（复用 EpubThemePalette） ───
// 从 domain/epub/types.ts 重导出
export type { EpubThemePalette } from "../domain/epub/types";

// ─── 阅读偏好（复用） ───
export type {
  ReaderPreferences,
  ReaderReadingMode,
  ReaderBackgroundTone,
  ReaderFontFamily,
} from "../utils/book-reader";

// ─── 引擎配置 ───
export interface EngineOptions {
  preferences: ReaderPreferences;
  theme: EpubThemePalette;
}
```

## 验收标准

1. `src/engine/types.ts` 文件创建完成，所有类型均有 JSDoc 注释。
2. 类型只做纯定义，不包含任何运行时实现代码。
3. 不引入新的第三方依赖。
4. `npm run build` 编译通过。
5. 现有测试不受影响。

## 预计改动量

- 新增 1 个文件，约 180–220 行。
