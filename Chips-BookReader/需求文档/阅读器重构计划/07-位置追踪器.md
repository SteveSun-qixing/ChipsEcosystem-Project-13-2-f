# 阶段 07 — 位置追踪器

## 目标

实现独立的阅读位置追踪模块，负责阅读位置的记录、锚点保存/恢复、窗口缩放后的位置重定位。

## 背景

当前位置恢复逻辑存在于 `ReadiumDocumentController` 的 `captureProgressionRatio` / `restoreProgressionRatio` / `capturePaginatedAnchor` / `restorePaginatedAnchor` 方法中。使用全局进度比例 (ratio) 做位置恢复的策略在窗口缩放导致列数变化时会产生跳页。

### 新策略

使用 **spread 索引 + 总 spread 数** 做位置锚点，而非全局像素比例：
- 保存时记录当前 `spreadIndex` 和 `totalSpreads`。
- 恢复时计算 `newSpreadIndex = Math.round(oldSpreadIndex / oldTotalSpreads * newTotalSpreads)`。
- 这样即使窗口缩放导致列数从 2 变为 1（spread 数翻倍），也能恢复到大致相同的内容位置。

### 锚点保存策略

```typescript
function captureAnchor(document: Document, mode: ReadingMode): ReadingPositionAnchor {
  if (mode === "paginated") {
    const scrollEl = resolveScrollingElement(document);
    const spreadWidth = scrollEl.offsetWidth;
    const currentOffset = Math.abs(scrollEl.scrollLeft);
    const maxOffset = scrollEl.scrollWidth - spreadWidth;
    const spreadIndex = spreadWidth > 0 ? Math.round(currentOffset / spreadWidth) : 0;
    const totalSpreads = spreadWidth > 0 ? Math.ceil(scrollEl.scrollWidth / spreadWidth) : 1;
    
    return {
      spreadIndex,
      spreadFraction: totalSpreads > 1 ? spreadIndex / (totalSpreads - 1) : 0,
      scrollFraction: maxOffset > 0 ? currentOffset / maxOffset : 0,
    };
  }
  
  // scroll mode
  const scrollEl = resolveScrollingElement(document);
  const scrollTop = scrollEl.scrollTop;
  const maxScroll = scrollEl.scrollHeight - scrollEl.clientHeight;
  
  return {
    spreadIndex: 0,
    spreadFraction: 0,
    scrollFraction: maxScroll > 0 ? scrollTop / maxScroll : 0,
  };
}
```

### 锚点恢复策略

```typescript
function restoreAnchor(
  anchor: ReadingPositionAnchor,
  document: Document,
  mode: ReadingMode,
): number {
  if (mode === "paginated") {
    const scrollEl = resolveScrollingElement(document);
    const newSpreadWidth = scrollEl.offsetWidth;
    const newTotalSpreads = newSpreadWidth > 0
      ? Math.ceil(scrollEl.scrollWidth / newSpreadWidth)
      : 1;
    const newMaxOffset = scrollEl.scrollWidth - newSpreadWidth;
    
    // 优先使用 spreadFraction 映射到新的 spread 索引
    const newSpreadIndex = Math.round(anchor.spreadFraction * (newTotalSpreads - 1));
    const targetOffset = Math.min(newSpreadIndex * newSpreadWidth, newMaxOffset);
    
    return Math.max(0, targetOffset);
  }
  
  // scroll mode
  const scrollEl = resolveScrollingElement(document);
  const newMaxScroll = scrollEl.scrollHeight - scrollEl.clientHeight;
  
  return Math.max(0, Math.round(anchor.scrollFraction * newMaxScroll));
}
```

### 全书进度计算

```typescript
function computeProgress(
  sectionIndex: number,
  sectionCount: number,
  sectionTitle: string,
  sectionSizes?: number[],
): ReadingProgress {
  const anchor = this.getCurrentAnchor();
  
  // 章节内进度
  const sectionFraction = anchor.spreadFraction;
  
  // 全书进度（简单线性：考虑章节权重）
  let bookFraction: number;
  if (sectionSizes && sectionSizes.length === sectionCount) {
    const totalSize = sectionSizes.reduce((sum, s) => sum + s, 0);
    const beforeSize = sectionSizes.slice(0, sectionIndex).reduce((sum, s) => sum + s, 0);
    const currentSize = sectionSizes[sectionIndex] ?? 0;
    bookFraction = totalSize > 0
      ? (beforeSize + currentSize * sectionFraction) / totalSize
      : sectionIndex / Math.max(1, sectionCount);
  } else {
    // 无权重，均匀分配
    bookFraction = (sectionIndex + sectionFraction) / Math.max(1, sectionCount);
  }
  
  // 页码（仅分页模式有意义）
  const currentPage = anchor.spreadIndex + 1;
  const totalPages = /* 从 PaginationMetrics 获取 */ 0;
  
  return {
    sectionIndex,
    sectionCount,
    sectionTitle,
    sectionFraction: Math.max(0, Math.min(1, sectionFraction)),
    bookFraction: Math.max(0, Math.min(1, bookFraction)),
    currentPage,
    totalPages,
    displayLabel: `${sectionIndex + 1} / ${sectionCount}`,
  };
}
```

## 验收标准

1. 分页模式下窗口缩放（宽→窄、窄→宽）后，阅读位置正确恢复到相同内容区域。
2. 双页→单页切换（列数变化）后位置恢复正确。
3. 滚动模式下窗口缩放后滚动位置正确恢复。
4. 全书进度计算准确，0–1 之间线性增长。
5. 章节内进度计算准确。
6. 所有纯函数有单元测试。
