# 阶段 14 — 搜索功能

## 目标

实现电子书内容全文搜索功能，支持跨章节搜索、结果高亮、结果列表和跳转。

## 背景

搜索是阅读器的重要辅助功能。当前完全缺失。

### 设计约束

- 搜索范围：全部章节的文本内容。
- 搜索方式：纯文本匹配（不做模糊匹配、不做正则表达式）。
- 高亮方式：搜索结果在当前章节的 iframe 中通过 DOM 操作高亮。
- 性能：搜索在后台逐章节进行，每找到一批结果就增量更新 UI，不阻塞主线程。

## 涉及文件

### 新建

export interface SearchResult {
  sectionIndex: number;
  sectionTitle: string;
  excerpt: string;           // 匹配文本上下文（前后各 40 字符）
  matchOffset: number;       // 匹配在章节文本中的偏移
  matchLength: number;       // 匹配文本长度
}

export class SearchEngine {
  constructor(private book: EpubBook);

### 增量搜索实现

```typescript
async *search(query: string): AsyncGenerator<SearchResult[]> {
  const normalizedQuery = query.trim().toLowerCase();
  if (!normalizedQuery) return;
  
  for (let i = 0; i < this.book.sections.length; i++) {
    // 检查取消
    if (this.cancelled) return;
    
    const section = this.book.sections[i];
    
    // 读取章节文本
    let text: string;
    try {
      const html = await this.book.archive.readText(section.path);
      text = this.extractPlainText(html);
    } catch {
      continue;
    }
    
    // 搜索匹配
    const results: SearchResult[] = [];
    const normalizedText = text.toLowerCase();
    let offset = 0;
    
    while (true) {
      const matchIndex = normalizedText.indexOf(normalizedQuery, offset);
      if (matchIndex < 0) break;
      
      results.push({
        sectionIndex: i,
        sectionTitle: section.title,
        excerpt: this.buildExcerpt(text, matchIndex, normalizedQuery.length),
        matchOffset: matchIndex,
        matchLength: normalizedQuery.length,
      });
      
      offset = matchIndex + 1;
    }
    
    if (results.length > 0) {
      yield results;
    }
    
    // 每章搜索完后让出主线程
    await new Promise(resolve => setTimeout(resolve, 0));
  }
}
```

### DOM 高亮实现

搜索结果高亮通过在 iframe 文档中动态插入 `<mark>` 标签实现：

```typescript
highlightInDocument(document: Document, results: SearchResult[], currentIndex: number): void {
  this.clearHighlights(document);
  
  // 遍历文本节点，找到匹配位置并包裹 <mark>
  const walker = document.createTreeWalker(document.body, NodeFilter.SHOW_TEXT);
  // ... 匹配和包裹逻辑
  
  // 当前选中的结果使用不同颜色
  const marks = document.querySelectorAll("mark[data-search-result]");
  marks[currentIndex]?.classList.add("search-current");
  marks[currentIndex]?.scrollIntoView({ block: "center" });
}
```

## 验收标准

1. 搜索框输入文本后可触发搜索。
2. 搜索结果增量显示，按章节分组。
3. 搜索过程不阻塞 UI。
4. 点击搜索结果可跳转到对应章节和位置。
5. 当前章节的搜索匹配在 iframe 中高亮显示。
6. 关闭搜索面板后高亮清除。
7. 空搜索和无结果有正确提示。
8. 文案全部走 i18n。
