# 阶段 15 — 样式重构与动画优化

## 目标

将 570 行的 `global.css` 拆分为多个职责明确的样式模块，并增加高品质的 UI 过渡动画。

## 背景

当前所有样式都写在一个 `global.css` 文件中，包括全局重置、空态样式、阅读器壳层、控制层、面板、按钮、导航、表单、反馈等全部样式。

此外，UI 过渡动画不够精致：
- 面板打开/关闭没有动画。
- 控制层显隐没有淡入淡出。
- 反馈消息没有入场/退场动画。

## 涉及文件

### 新建

- `src/styles/global.css` — 全局基础（重置、变量、字体）
- `src/styles/reader-shell.css` — 阅读器壳层、空态、视口
- `src/styles/reader-chrome.css` — 控制层（标题卡、导航、坞栏）
- `src/styles/reader-panels.css` — 面板（导入、目录、偏好、搜索、书签）
- `src/styles/reader-progress.css` — 进度条
- `src/styles/reader-animation.css` — 过渡动画

### 替换

- `src/global.css` → 归档到 `归档/`，由新的样式模块替代

## 动画优化

### 控制层淡入淡出

```css
.book-reader-chrome {
  opacity: 0;
  transform: translateY(8px);
  transition: opacity 240ms ease, transform 240ms ease;
  pointer-events: none;
}

.book-reader-chrome[data-state="visible"] {
  opacity: 1;
  transform: translateY(0);
  pointer-events: auto;
}
```

### 面板滑入滑出

```css
.book-reader-panelLayer {
  opacity: 0;
  transition: opacity 200ms ease;
}

.book-reader-panelLayer[data-state="open"] {
  opacity: 1;
}

.book-reader-panel--drawer {
  transform: translateX(100%);
  transition: transform 320ms cubic-bezier(0.32, 0.72, 0, 1);
}

.book-reader-panelLayer[data-state="open"] .book-reader-panel--drawer {
  transform: translateX(0);
}

.book-reader-panel--source {
  transform: scale(0.96);
  opacity: 0;
  transition: transform 280ms cubic-bezier(0.32, 0.72, 0, 1), opacity 200ms ease;
}

.book-reader-panelLayer[data-state="open"] .book-reader-panel--source {
  transform: scale(1);
  opacity: 1;
}
```

### 反馈消息入场

```css
.book-reader-feedback {
  animation: feedback-enter 320ms cubic-bezier(0.32, 0.72, 0, 1);
}

@keyframes feedback-enter {
  from {
    opacity: 0;
    transform: translateX(-50%) translateY(12px);
  }
  to {
    opacity: 1;
    transform: translateX(-50%) translateY(0);
  }
}
```

### 进度条动画

```css
.book-reader-progress__fill {
  transition: width 160ms ease;
}

.book-reader-progress__thumb {
  transition: left 160ms ease, transform 120ms ease;
}

.book-reader-progress__thumb:active {
  transform: translateX(-50%) scale(1.4);
}
```

### 坞栏按钮入场

```css
.book-reader-chrome__dock {
  animation: dock-enter 360ms cubic-bezier(0.32, 0.72, 0, 1);
}

@keyframes dock-enter {
  from {
    opacity: 0;
    transform: translateX(-50%) translateY(16px);
  }
  to {
    opacity: 1;
    transform: translateX(-50%) translateY(0);
  }
}
```

## 验收标准

1. 所有视觉效果与旧版保持一致或更优。
2. 面板打开/关闭有流畅的滑入/滑出动画。
3. 控制层显隐有淡入淡出动画。
4. 反馈消息有入场动画。
5. 进度条拖拽有流畅的跟随效果。
6. 动画遵循 `prefers-reduced-motion` 媒体查询。
7. 样式文件总行数不超过旧 global.css 的 1.5 倍。
8. 所有样式使用主题 token（`--chips-sys-color-*`）。
9. 响应式断点覆盖移动端和桌面端。
10. `npm run build` 后样式正常生效。
