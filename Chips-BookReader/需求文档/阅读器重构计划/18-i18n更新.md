# 阶段 18 — i18n 更新

## 目标

为新增的 UI 功能补全多语言文案。

## 背景

重构新增了进度条、书签面板、搜索面板等 UI 功能，需要在 `zh-CN.json` 和 `en-US.json` 中补充对应的文案。

## 涉及文件

### 修改

- `i18n/zh-CN.json`
- `i18n/en-US.json`

## 新增文案清单

### 进度相关

| Key | zh-CN | en-US |
|-----|-------|-------|
| `book-reader.labels.progress` | 阅读进度 | Reading Progress |
| `book-reader.reader.pagePosition` | 第 {current} / {total} 页 | Page {current} / {total} |
| `book-reader.reader.bookProgress` | 全书 {percentage}% | {percentage}% complete |

### 书签相关

| Key | zh-CN | en-US |
|-----|-------|-------|
| `book-reader.labels.bookmarks` | 书签 | Bookmarks |
| `book-reader.actions.addBookmark` | 添加书签 | Add Bookmark |
| `book-reader.actions.removeBookmark` | 删除书签 | Remove Bookmark |
| `book-reader.actions.toggleBookmark` | 切换书签 | Toggle Bookmark |
| `book-reader.actions.openBookmarks` | 打开书签 | Open Bookmarks |
| `book-reader.bookmarks.empty` | 还没有书签 | No bookmarks yet |
| `book-reader.bookmarks.count` | 共 {count} 个书签 | {count} bookmark(s) |
| `book-reader.bookmarks.pageLabel` | 第 {page} 页 | Page {page} |
| `book-reader.bookmarks.scrollLabel` | {percentage}% 处 | At {percentage}% |

### 搜索相关

| Key | zh-CN | en-US |
|-----|-------|-------|
| `book-reader.labels.search` | 搜索 | Search |
| `book-reader.actions.search` | 搜索电子书 | Search Book |
| `book-reader.actions.clearSearch` | 清除搜索 | Clear Search |
| `book-reader.placeholders.search` | 搜索电子书内容... | Search book content... |
| `book-reader.search.results` | 找到 {count} 个结果 | {count} result(s) found |
| `book-reader.search.noResults` | 没有找到匹配的内容 | No matches found |
| `book-reader.search.searching` | 正在搜索 第 {current} / {total} 章... | Searching section {current} / {total}... |
| `book-reader.search.matchesInSection` | {count} 个匹配 | {count} match(es) |

### 控件相关

| Key | zh-CN | en-US |
|-----|-------|-------|
| `book-reader.actions.toggleSearch` | 搜索 | Search |

## 验收标准

1. 所有新增 UI 文案在 zh-CN 和 en-US 中都有对应条目。
2. 插值参数语法正确（`{key}` 格式）。
3. JSON 格式有效。
4. 无硬编码文案残留在组件中。

## 预计改动量

- 修改 2 个文件，各增加约 25–30 行。
