# Chips-CardViewer AGENTS

## 角色

- 本项目是卡片查看器应用插件，是 Host 统一卡片显示链路的消费方和应用套壳，不是渲染运行时本体。

## 开发规则

- 卡片显示必须通过 SDK / Host 的正式封面、复合卡片和启动上下文链路完成，不得在本项目内重写基础卡片分发、模板编译或 iframe 拼接。
- 主题、语言、窗口背景和卡片内容区必须统一接入正式运行时，响应 `theme.changed`、`language.changed` 等正式事件。
- 文件打开、快捷方式和文件关联必须遵循正式 `launchContext` 与插件启动契约，不要在应用内私建第二套入口协议。
- 错误态、加载态和日志必须保留可诊断信息；如果显示链路公共契约有变动，应同步更新共用文档。

## 验证

- 优先运行本项目 `build/test/lint/validate` 等正式脚本。
