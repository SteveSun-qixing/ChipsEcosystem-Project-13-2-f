# 阶段四：工作区上传页实现

## 1. 阶段目标

- 实现拖拽上传工作区；
- 打通卡片与箱子上传反馈；
- 形成清晰的结果闭环。

## 2. 实施内容

1. 搭建全页拖拽区；
2. 建立统一上传队列状态模型；
3. 接通 `.card` / `.box` 自动识别上传；
4. 实现卡片内容 HTML 与封面 HTML 双产物处理链路；
5. 实现卡片处理状态轮询；
6. 实现成功 / 失败 / 处理中反馈；
7. 实现返回个人页和打开内容入口。

## 3. 关键注意项

- 不恢复旧版“模式切换上传中心”；
- 不加入房间选择与可见性切换；
- 队列信息必须简洁但完整。

## 4. 阶段验收

- 拖拽和点选都可用；
- 卡片、箱子两条链路都可正确反馈；
- 工作区页面不演变成新的后台工作台。
