# 阶段五：视觉动效与体验收口

## 1. 阶段目标

- 把页面从“可用”打磨到“有质感”；
- 完成悬浮感、漂动感、细腻过渡与响应式收口。

## 2. 实施内容

1. 调整头图区材质、留白和层级；
2. 调整作品卡片阴影、圆角、光泽与 hover；
3. 实现轻微随机漂动；
4. 收口上传页拖拽热区反馈与队列转场；
5. 增加 `prefers-reduced-motion` 降级。

## 3. 质量门槛

- 动效细腻，不喧宾夺主；
- 所有交互过渡平滑；
- 移动端不崩布局；
- 主题切换后质感一致。

## 4. 阶段验收

- 作品封面具备轻微悬浮和漂动感；
- 页面对比旧版明显更简洁；
- 没有多余模块残留。
