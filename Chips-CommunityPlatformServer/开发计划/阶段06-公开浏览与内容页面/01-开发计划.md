# 阶段六：公开浏览与内容页面

**阶段编号**：Phase 6  
**前置阶段**：Phase 5（箱子功能完成）  
**预计工时**：2-3 天  
**目标**：面向所有互联网用户的完整内容浏览体验。

---

## 交付标准

- 首页展示最新公开卡片和箱子
- 支持关键词搜索
- 卡片详情页完整展示（HTML iframe）
- 个人空间 / 房间页面内容填充完整
- 基础 SEO 完善（title、meta description、Open Graph）

---

## 关键任务

### 服务端

#### 发现接口（`src/routes/discover.ts`）

- [ ] `GET /api/v1/discover/cards?page=1&pageSize=20` → 最新公开卡片（status=ready）
- [ ] `GET /api/v1/discover/boxes?page=1&pageSize=20` → 最新公开箱子
- [ ] `GET /api/v1/search?q=关键词&type=card,box,user` → 全文搜索（PostgreSQL pg_trgm）

### 前端

#### HomePage（`src/pages/HomePage.tsx`）

- [ ] 顶部搜索栏（输入 → `/search?q=...`）
- [ ] 最新卡片网格（无限滚动 or 分页）：封面 + 标题 + 用户名
- [ ] 最新箱子网格

#### CardDetailPage（`src/pages/CardDetailPage.tsx`）

- [ ] 面包屑：首页 → @用户名 → 卡片标题
- [ ] 主体区：将 `html_url` 嵌入 `<iframe sandbox="allow-scripts allow-same-origin">`
  - iframe 高度自适应（监听 `postMessage` 或使用 ResizeObserver）
- [ ] 侧栏：卡片元数据（基础卡片类型列表、标签、创建时间、文件大小）
- [ ] SEO：`<title>{cardName} - 薯片社区</title>`，`<meta name="description">` 取 card 名 + 用户名

#### BoxDetailPage 完善

- [ ] 更完整的 SEO 处理
- [ ] Open Graph tags（`og:title`、`og:image`=封面、`og:description`）

#### SearchResultPage（`src/pages/SearchResultPage.tsx`）

- [ ] Tab 切换：卡片 / 箱子 / 用户
- [ ] 搜索结果列表（含分页）
- [ ] 空结果状态

#### SpacePage 和 RoomPage 完整化

- [ ] SpacePage：根目录卡片与箱子填充（调用 `/users/:username/space`）
- [ ] RoomPage：展示房间内卡片和箱子列表（调用 `/rooms/:roomId/contents`）
