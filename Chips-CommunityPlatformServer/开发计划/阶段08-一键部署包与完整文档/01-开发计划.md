## 关键任务

### 生产 Docker Compose 完善

- [ ] `deploy/docker-compose.prod.yml`：
  - 所有服务使用 named volume 持久化
  - 设置 `restart: unless-stopped`
  - Nginx 配置 SSL（模板，支持 Let's Encrypt Certbot）
  - 设置资源限制（memory limits）
- [ ] Nginx 配置完善：
  - HTTPS 重定向
  - 静态文件 gzip 压缩
  - 合适的缓存头（资源文件长缓存，HTML 文件短缓存）
  - 安全头（X-Frame-Options、X-Content-Type-Options 等）
- [ ] `deploy/init-ssl.sh`：Let's Encrypt 自动申请脚本模板

### Swagger / OpenAPI 文档

- [ ] 安装 `@fastify/swagger`、`@fastify/swagger-ui`（MIT 协议，需确认）
- [ ] 所有路由配置 schema（request / response）
- [ ] Swagger UI 在 `/api/docs` 可访问（生产环境可通过 env 关闭）

### 部署文档（`deploy/docs/`）

## 部署架构图（最终状态）

```
Internet
    │
    ▼
Nginx (443/80)
    ├── /* → Web 前端静态文件（React 构建产物）
    ├── /admin/* → Admin 后台静态文件
    ├── /api/* → Fastify API (port 3000 内部)
    └── /cdn/* → MinIO (port 9000 内部)
         
内部网络：
  Fastify ← PostgreSQL (5432)
  Fastify ← Redis (6379)
  Fastify ← MinIO (9000)
  
数据持久化：
  PostgreSQL → postgres-data volume
  MinIO → minio-data volume
  Redis → redis-data volume
```
