# 社区前端 V2 技术手册索引

## 文档定位

- 本目录用于描述 `Chips-CommunityPlatformServer/packages/web` 的社区前端 V2 技术方案；
- 本轮技术目标是完成旧壳层收敛、路由重组、作品聚合页重做、工作区上传页重做；
- 本手册同样是**目标设计口径**，不是对旧版前端继续沿用的认可。
- 其中 `07-启动与运行说明.md` 是面向整个 `Chips-CommunityPlatformServer` 项目的运行与联调补充文档，以当前 `server / web / admin / deploy` 实现为准。
