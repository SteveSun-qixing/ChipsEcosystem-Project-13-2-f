# 管理后台产品规则

## 1. 产品定位

管理后台是平台运营入口，不是面向普通用户的工作台。

第一版目标是提供最基础但完整可用的治理能力：

- 看平台规模
- 管用户状态
- 管社区内容

## 2. 访问规则

### 2.1 登录规则

后台当前使用普通登录接口，不维护第二套后台认证体系。

### 2.2 角色规则

只有 `role = admin` 的用户可以进入后台功能区。

因此后台访问判定分三种状态：

1. 未登录
2. 已登录但不是管理员
3. 已登录且是管理员

## 3. 后台功能板块

### 3.1 平台总览

当前展示：

- 总用户数
- 总卡片数
- 总箱子数
- 总存储量
- 今日新增用户数
- 今日新增卡片数
- 今日新增箱子数

### 3.2 用户管理

当前支持：

- 搜索用户名 / 显示名称
- 分页浏览用户列表
- 查看角色
- 查看启用状态
- 查看存储占用
- 启用 / 禁用用户

### 3.3 内容治理

当前支持：

- 卡片 / 箱子切换
- 按标题搜索
- 分页浏览
- 查看基础元数据
- 强制删除内容

## 4. 当前后台交互规则

### 4.1 搜索

- 用户搜索与内容搜索各自独立
- 搜索提交后回到第 1 页

### 4.2 分页

- 用户列表与内容列表各自独立分页
- 分页切换时保留当前查询条件

### 4.3 删除确认

内容删除采用“二次点击确认”规则，避免误删。

### 4.4 用户启停

用户启用 / 禁用为即时操作，提交后刷新列表。

## 5. 后台当前展示字段

### 5.1 用户列表字段

- 用户名 / 显示名称
- 角色
- 状态
- 存储占用
- 创建时间
- 操作按钮

### 5.2 内容列表字段

卡片 / 箱子当前会展示：

- 标题
- 所属用户 ID
- 可见性
- 文件大小
- 创建时间
- 更新时间
- 卡片状态或箱子布局插件（按类型不同）
- 删除操作

## 6. 主题与多语言

后台当前内置：

- 中文 / 英文
- 纸页 / 夜幕两套主题

这些能力当前由后台项目内部自行维护。

## 7. 当前范围外

第一版后台暂不包含：

- 审计日志页面
- 举报处理流
- 平台公告发布
- 角色细分配置
- 数据导出中心
- 图表分析看板

## 8. 第一版后台验收要求

后台至少要满足：

1. 管理员能稳定登录
2. 能看到平台统计数据
3. 能搜索并启停用户
4. 能搜索并删除卡片 / 箱子
5. 非管理员不能进入后台功能区
