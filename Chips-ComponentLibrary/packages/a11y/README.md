# @chips/a11y

Accessibility helper functions.

## APIs

- keyboard:
  - `isKeyboardActivationKey`
  - `isKeyboardNavigationKey`
  - `getKeyboardIntent`
- aria:
  - `createAriaStatusProps`
  - `buildAriaDescribedBy`
  - `validateAriaProps`
  - `assertAriaProps`
- focus:
  - `isFocusableElement`
  - `getFocusableElements`
  - `moveFocus`
