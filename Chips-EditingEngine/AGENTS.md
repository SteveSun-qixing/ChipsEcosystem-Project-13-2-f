# Chips-EditingEngine AGENTS

## 角色

- 本项目是编辑引擎正式应用工程；涉及基础卡片装配、编辑运行时、工作区、窗口、资源、持久化、主题和多语言接线。

## 开发规则

- 编辑链路必须优先使用正式的基础卡片运行时、Host / Bridge / SDK 服务和当前工程的真实业务代码，不得私建一套旁路运行时。
- 工作区、文件、资源、设置、主题和多语言能力统一经现有 `services/`、Bridge 和 SDK 正式入口访问，不要绕层直连。
- 任何影响窗口、工具面板、基础卡片装配、保存链路、预览链路或资源生命周期的改动，都要同步检查主题、多语言、持久化和测试。
- 当前如果 README、模板残留或示例文件与真实 `manifest.yaml`、代码和内部技术文档冲突，以真实代码、`manifest.yaml` 和正式内部文档为准，并补记工单。
- 编辑引擎对外公开的生态规则、显示协议和公共编辑契约，应沉淀到 `生态共用技术文档/`，不要长期只留在本仓。

## 验证

- 优先运行本项目 `build/test/lint/validate` 等正式脚本，并补充与变更范围匹配的专项测试。
