# 模块改造清单

## 1. 需要重点修改的现有文件

### `src/components/CardWindow/CardWindow.tsx`

改造目标：

- 从 `client.card.compositeWindow.render(...)` 改为编辑引擎内部复合拼装；
- 负责装配各基础卡片 iframe；
- 继续维护窗口态、选中态、刷新节流和无限画布交互。

### `src/components/EditPanel/PluginHost.tsx`

改造目标：

- 从 `SDK editorPanel` iframe 宿主改为本地 `EditorHost`；
- 去掉 `getChipsClient().card.editorPanel` 依赖；
- 接入本地会话层、注册中心、校验和提交。

### `src/components/EditPanel/EditPanel.tsx`

改造目标：

- 从“直接把卡片数据塞给 iframe 宿主”改为“解析本地编辑器运行时上下文”；
- 补齐空态、错误态、编辑器不可用态。

### `src/core/card-service.ts`

改造目标：

- 保持单一数据源；
- 强化 `updateBasicCard` 相关事件和卡片类型规范化逻辑；
- 为本地编辑器会话提供更稳定的数据输入输出边界。

### `src/context/CardContext.tsx`

改造目标：

- 根据编辑器运行时需要补充更清晰的 selector/操作接口；
- 避免 `EditPanel` 直接在组件层做过多卡片解析逻辑。
