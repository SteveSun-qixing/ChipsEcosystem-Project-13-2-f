# chips-scaffold-module AGENTS

## 角色

- 生成模块插件正式模板；必须与当前冻结的模块插件契约、Host / SDK 代码和共享文档保持一致。

## 开发规则

- 不要把未冻结或仍有分歧的 `module.*` 契约写进模板；若 Host、SDK 和文档口径不一致，先登记工单。
- 生成工程必须自带正式目录、测试、文档骨架和 CLI 接入所需结构，不能让业务仓补模板基线。
- 模块模板的公共规范变化要同步共享文档，避免模板和正式规范分叉。
- 参数渲染器、模板目录、CLI 集成和生成工程校验要一起维护。

## 验证

- 优先运行 `build`、`test`、`test:templates`、`test:e2e`。
