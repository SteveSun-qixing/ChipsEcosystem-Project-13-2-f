# 阶段二：标准模块插件模板骨架实现

## 1. 阶段目标

- 建立 `chips-scaffold-module` 包；
- 实现模板引擎、错误模型、文件系统工具与 CLI API；
- 落地 `module-standard` 标准模板；
- 让模板在默认状态下具备真实可运行的 React + SDK + 主题/i18n 示例。

## 2. 主要工作项

- 新建 `src/core` 与 `src/cli`；
- 定义 `CreateModuleProjectOptions` 与模板元数据结构；
- 实现模板渲染引擎；
- 补齐标准模板中的：
  - Manifest
  - 构建配置
  - React 运行时
  - 词典与日志
  - 单元测试
  - README

## 3. 阶段完成标准

- `npm run build` 与 `vitest` 可通过；
- 模板渲染单测可通过；
- 生成工程具备正式结构与基础质量门禁能力。
