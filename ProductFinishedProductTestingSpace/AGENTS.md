# ProductFinishedProductTestingSpace AGENTS

## 角色

- 本目录是生态工作区级测试与成品验证空间，不是普通业务子项目。
- 这里主要保存工作区配置和验证样例，不承载某个应用的正式源码实现。

## 开发规则

- 修改这里时优先考虑整个生态工作区的一致性、依赖基线和验证链路，不要写只服务单个项目的私有规则。
- 工作区级配置变化要同步核对根 `package.json`、工作区模式和 `scripts/verify-ecosystem-workspace.mjs`。
- 示例卡片或测试材料应服务于正式验证，不要把公共契约文档长期堆在这里。

## 验证

- 优先运行工作区级验证脚本和与改动相关的总仓验证。
