### P4-06：实现快捷键

**描述**：实现键盘快捷键支持

**快捷键列表**：
| 快捷键 | 功能 |
|--------|------|
| Ctrl/Cmd+B | 加粗 |
| Ctrl/Cmd+I | 斜体 |
| Ctrl/Cmd+U | 下划线 |
| Ctrl/Cmd+Z | 撤销 |
| Ctrl/Cmd+Shift+Z | 重做 |
| Tab | 列表缩进 |
| Shift+Tab | 列表减少缩进 |

**实现位置**：
- 文本格式快捷键：EditorContent.vue
- 撤销重做快捷键：RichTextEditor.ts（全局监听）

**验收标准**：
