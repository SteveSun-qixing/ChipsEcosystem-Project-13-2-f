## 1. 架构概述

### 1.1 定位

富文本基础卡片插件是薯片生态中的一个标准基础卡片插件，遵循薯片插件开发规范。插件包含完整的渲染组件和编辑组件，可被编辑引擎、查看器等产品调用。

### 1.2 架构原则

1. **中心路由通信**：所有外部通信通过薯片内核路由
2. **功能样式分离**：组件只包含功能逻辑，样式由主题包提供
3. **独立可替换**：插件可独立安装、更新、卸载
4. **轻量简单**：只实现核心功能，不过度设计

## 5. 关键算法

### 5.1 HTML安全过滤算法

```typescript
function sanitizeHtml(html: string): string {
  // 1. 解析为DOM
  const parser = new DOMParser();
  const doc = parser.parseFromString(html, 'text/html');
  
  // 2. 遍历所有节点
  walkNodes(doc.body, (node) => {
    if (node.nodeType === Node.ELEMENT_NODE) {
      const element = node as Element;
      
      // 3. 检查标签白名单
      if (!ALLOWED_TAGS.includes(element.tagName.toLowerCase())) {
        // 替换为内容
        element.replaceWith(...element.childNodes);
        return;
      }
      
      // 4. 移除危险属性
      for (const attr of element.attributes) {
        if (!isAllowedAttribute(element.tagName, attr.name)) {
          element.removeAttribute(attr.name);
        }
        // 检查事件处理器
        if (attr.name.startsWith('on')) {
          element.removeAttribute(attr.name);
        }
      }
      
      // 5. 检查链接协议
      if (element.tagName === 'A') {
        const href = element.getAttribute('href');
        if (!isAllowedProtocol(href)) {
          element.removeAttribute('href');
        }
      }
    }
  });
  
  return doc.body.innerHTML;
}
```

### 5.2 撤销重做算法

```typescript
class UndoManager {
  private history: HistoryEntry[] = [];
  private position: number = -1;
  private maxSize: number = 100;
  
  push(entry: HistoryEntry): void {
    // 清除position之后的历史
    this.history = this.history.slice(0, this.position + 1);
    
    // 添加新记录
    this.history.push(entry);
    this.position++;
    
    // 限制大小
    if (this.history.length > this.maxSize) {
      this.history.shift();
      this.position--;
    }
  }
  
  undo(): HistoryEntry | null {
    if (this.position < 0) return null;
    const entry = this.history[this.position];
    this.position--;
    return entry;
  }
  
  redo(): HistoryEntry | null {
    if (this.position >= this.history.length - 1) return null;
    this.position++;
    return this.history[this.position];
  }
}
```

### 5.3 格式化命令执行

```typescript
function executeFormat(command: FormatCommand): void {
  // 获取当前选区
  const selection = window.getSelection();
  if (!selection || selection.rangeCount === 0) return;
  
  const range = selection.getRangeAt(0);
  
  switch (command.type) {
    case 'bold':
      document.execCommand('bold', false);
      break;
      
    case 'italic':
      document.execCommand('italic', false);
      break;
      
    case 'heading':
      if (command.level === 0) {
        document.execCommand('formatBlock', false, 'p');
      } else {
        document.execCommand('formatBlock', false, `h${command.level}`);
      }
      break;
      
    case 'color':
      document.execCommand('foreColor', false, command.value);
      break;
      
    // ... 其他命令
  }
  
  // 记录到撤销历史
  undoManager.push({
    type: 'format',
    command,
    range: range.cloneRange()
  });
}
```

## 6. 性能优化

### 6.1 渲染优化

1. **懒加载图片**：使用IntersectionObserver延迟加载
2. **虚拟DOM**：利用Vue的虚拟DOM减少实际DOM操作
3. **防抖渲染**：输入时使用防抖延迟渲染

### 6.2 编辑优化

1. **节流保存**：自动保存使用节流避免频繁触发
2. **选区缓存**：缓存选区计算结果
3. **增量更新**：只更新变化的DOM部分

### 6.3 内存优化

1. **及时清理**：销毁时清理所有引用
2. **事件解绑**：移除所有事件监听器
3. **历史限制**：撤销历史有最大数量限制

### 7.2 错误处理策略

```typescript
try {
  await this.render(config, container, options);
} catch (error) {
  if (error instanceof ChipsError) {
    // 已知错误，记录日志并显示友好提示
    Logger.error(`RichText error: ${error.code}`, error);
    this.showError(error.message);
  } else {
    // 未知错误，记录详细信息
    Logger.error('Unknown RichText error', error);
    this.showError(t('error.unknown'));
  }
}
```

## 8. 安全设计

### 8.1 XSS防护

1. **白名单标签**：只允许安全的HTML标签
2. **属性过滤**：移除所有事件处理器属性
3. **协议限制**：链接只允许http/https/mailto
4. **样式限制**：只允许安全的CSS属性

### 8.2 白名单配置
