## 📝 改进说明

### 多语言系统
现在所有文本都通过 `t()` 函数获取：
```typescript
// 开发时
t('toolbar.bold')  // → "加粗"

// 构建后（系统自动替换）
t('i18n.plugin.400001')  // → 根据当前语言返回翻译
```

### 错误处理
使用标准错误类：
```typescript
throw new ResourceError(
  RichTextErrorCode.FILE_NOT_FOUND,
  t('error.file_not_found', { file: fileName }),
  { file: fileName }
);
```

### 服务注册
在初始化时注册服务：
```typescript
await this.registerServices();
// - richtext.render
// - richtext.format
```
