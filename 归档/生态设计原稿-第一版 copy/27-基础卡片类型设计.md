# 基础卡片类型设计

(基础卡片不要进行分类)

富文本卡片用于显示rtf格式的文本内容。数据模型包含内容来源（文件或内联）、内容文本、是否启用工具栏等字段。如果选择文件模式，记录富文本文件的路径。如果选择内联模式，文本内容直接保存在配置中。编辑时提供所见即所得的富文本编辑器，工具栏包含加粗、斜体、标题、列表、颜色等基础格式化功能。前端渲染时解析富文本格式，显示为格式化的文本，支持响应式布局。

Markdown 卡片用于显示 Markdown 格式的文档。数据模型包含 Markdown 来源（文件或内联）、内容文本、是否显示目录、是否启用代码高亮等字段。编辑时提供 Markdown 编辑区和实时预览，可以插入图片、链接等元素。前端渲染时解析 Markdown 语法转换为 HTML，支持代码高亮和目录生成。

代码块卡片用于显示代码片段。数据模型包含代码内容、编程语言类型、是否显示行号、主题等字段。编辑时提供代码编辑器，支持语法高亮和自动缩进。前端渲染时应用语法高亮，显示行号，提供复制按钮。

图片卡片用于展示单张图片。数据模型包含图片文件路径、标题说明、适配模式（填充、包含、裁剪）等字段。编辑时提供图片上传功能和标题输入框。前端渲染时显示图片，支持点击放大，支持缩放和旋转。

视频卡片用于播放视频。数据模型包含视频文件路径、封面图片路径、字幕列表（每个字幕包含文件路径、语言代码、标签）、是否自动播放、是否显示控制条等字段。编辑时提供视频文件上传、封面选择、字幕文件上传和管理功能。前端渲染时创建视频播放器，支持播放控制、字幕切换、进度记忆。

音乐卡片用于播放音频。数据模型包含音频文件路径、封面图片路径、歌词文件路径、是否显示音频可视化等字段。编辑时提供音频上传、封面选择、歌词上传功能。前端渲染时创建音乐播放器，显示播放控制、歌词同步显示、音频波形可视化。

3D模型卡片用于展示三维模型。数据模型包含模型文件路径、模型格式。编辑时提供模型文件上传。前端渲染时使用三维渲染引擎显示模型，支持鼠标旋转和缩放。

列表卡片用于显示各种类型的列表。数据模型包含列表类型（待办、有序、无序）、列表项数组（每项包含内容、是否完成等）、是否可编辑等字段。编辑时提供列表项的添加、删除、排序功能，待办列表可以标记完成状态。前端渲染时根据类型显示对应样式的列表，待办列表显示复选框，有序列表显示序号。

打分卡片用于显示评分。数据模型包含分数值、最高分（五分制或十分制）、评分说明文本等字段。编辑时提供星星选择器或滑块设置分数。前端渲染时显示星星或进度条，显示评分说明。

网页卡片用于嵌入自定义的网页内容。数据模型包含 HTML 来源（文件或内联）、HTML 内容、是否允许脚本执行等字段。编辑时提供 HTML 编辑区或文件上传。前端渲染时在网页框架中显示 HTML 内容，根据安全设置决定是否允许脚本。

游戏卡片用于嵌入小游戏。数据模型包含游戏文件路径（HTML5 游戏包）、游戏配置等字段。编辑时上传游戏文件包。前端渲染时在网页框架中加载游戏，用户可以直接玩。

应用卡片用于嵌入小应用。数据模型包含应用文件路径、应用类型、配置参数等字段。编辑时上传应用包或选择已安装的应用。前端渲染时加载应用界面，用户可以与应用交互。

日历卡片用于显示日历。数据模型包含年月信息、事件列表（每个事件包含日期、时间、标题、描述）、显示模式（月视图、周视图、日视图）等字段。编辑时提供事件的添加、删除、日期选择功能。前端渲染时显示日历网格，事件标记在对应日期，点击日期查看详情。

一天卡片用于记录一天的内容。数据模型包含日期、时间段列表（每个时间段包含时间、事件、心情等）、天气、总结等字段。编辑时提供时间段的添加和编辑，可以设置心情图标。前端渲染时按时间轴显示一天的活动，不同时间段用不同颜色区分。

甘特图卡片用于项目管理。数据模型包含任务列表（每个任务包含名称、开始日期、结束日期、进度、依赖关系）、时间范围等字段。编辑时提供任务的添加、日期设置、依赖关系设置功能。前端渲染时显示甘特图，任务显示为时间条，可以拖动调整时间。

热力图卡片用于数据可视化。数据模型包含数据源（文件或内联数据）、数据格式、颜色方案、是否显示数值等字段。编辑时提供数据上传或手动输入。前端渲染时根据数据生成热力图，不同数值用不同颜色深浅表示。

思维导图卡片用于知识结构展示。数据模型包含节点列表（每个节点包含内容、父节点、位置、颜色）、布局方式（树状、放射状）等字段。编辑时提供节点的添加、连接、拖动排列功能。前端渲染时显示思维导图，节点用连线关联，支持展开折叠。

无限画布卡片用于自由创作。数据模型包含元素列表（每个元素包含类型、位置、大小、内容）、画布大小等字段。编辑时提供工具栏添加文本、图形、图片等元素，可以拖动和调整。前端渲染时显示画布，所有元素按位置排列，支持缩放和平移。

地图卡片用于显示地理位置。数据模型包含中心坐标、缩放级别、标记点列表（每个标记包含坐标、标题、描述）、地图类型等字段。编辑时提供地图选择和标记点添加功能。前端渲染时加载地图服务，显示地图和标记点，支持缩放和平移。

剧集卡片用于管理多集视频。数据模型包含剧集信息（标题、简介、封面）、分集列表（每集包含标题、视频路径、时长）等字段。编辑时提供剧集信息编辑和分集管理功能。前端渲染时显示剧集封面和分集列表，点击某集播放视频。

图文卡片用于图文混排内容。数据模型包含内容块列表（每个块可以是文本或图片）、排列方式等字段。编辑时提供添加文本块和图片块功能，可以调整顺序。前端渲染时按顺序显示文本和图片，支持图片点击放大。

帖子卡片用于社交帖子。数据模型包含标题、正文、图片列表、发布时间、点赞数、评论数等字段。编辑时提供标题和正文输入、图片上传功能。前端渲染时显示帖子样式，标题突出，图片网格排列，显示互动数据。

聊天记录卡片用于保存对话。数据模型包含对话列表（每条消息包含发送者、时间、内容、类型）、对话参与者等字段。编辑时提供消息的添加和角色设置。前端渲染时显示聊天气泡样式，不同角色的消息左右排列。

名片卡片用于展示个人或组织信息。数据模型包含姓名、头像、职位、公司、联系方式（电话、邮箱、地址）、社交账号等字段。编辑时提供表单输入和头像上传。前端渲染时显示名片样式，头像居中或靠左，信息分区显示。

商品卡片用于展示商品。数据模型包含商品名称、价格、图片列表、描述、规格参数等字段。编辑时提供商品信息编辑和图片上传。前端渲染时显示商品详情页样式，图片轮播，价格突出显示，参数表格展示。

果式设备卡片用于记录电子设备信息。数据模型包含设备类型、型号、购买日期、保修期、配置参数、照片等字段。编辑时提供设备信息表单填写。前端渲染时显示设备信息卡片，照片展示，参数列表显示。



编辑组件由对应的插件提供，插件开发者可以自行设计界面和交互方式。渲染组件也由插件提供，根据配置文件生成网页内容，在网页框架中显示。

每种基础卡片都可以组合成独立入口的应用插件产品。比如日历卡片能力可以组合成日历应用插件，商品卡片能力可以组合成商品管理应用插件，运行时统一由 Chips Host 承载。
