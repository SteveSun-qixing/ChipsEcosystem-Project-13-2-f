# 插件开发者快速入门

**版本**：1.0.0  
**最后更新**：2026-02-17  
**目标读者**：首次接触薯片生态的插件开发者

---

## 1. 5 分钟理解生态

1. 薯片生态只有一个宿主：`Chips Host`  
2. 你的应用以插件形式运行在 Host 内  
3. 插件只能通过 `window.chips.*`（或 `@chips/sdk`）访问系统能力  
4. 所有插件统一交付为 `.cpk`

---

## 2. 环境准备

```bash
node -v
pnpm -v
pnpm add -g @chips/develop-cli
chipsd --version
```

---

## 3. 创建第一个插件

```bash
chipsd create my-first-plugin
cd my-first-plugin
pnpm install
chipsd dev
```

建议优先从标准模板开始（app/card/layout/module/theme）。

---

## 4. 日常开发命令

```bash
# 本地开发
chipsd dev

# 构建
chipsd build

# 打包为 .cpk
chipsd pack

# 合规校验
chipsd validate
```

多语言相关：

```bash
chipsd i18n check
chipsd i18n extract
chipsd i18n generate --lang en-US
```

---

## 5. 必须遵守的开发规则

1. **中心路由原则**：主进程服务之间必须经内核路由通信  
2. **Bridge 隔离原则**：插件侧禁止直接使用 Node/Electron API  
3. **零硬编码文本**：用户可见文本必须走多语言系统  
4. **零样式组件**：组件不内置视觉样式，视觉由主题注入  
5. **TypeScript strict**：禁止无说明的 `any`

---

## 6. 发布前检查

1. `vitest` 全通过  
2. `chipsd validate` 全通过  
3. `.cpk` 可安装、可启动、可卸载  
4. Host 联调通过（不是只有 Mock 模式通过）
