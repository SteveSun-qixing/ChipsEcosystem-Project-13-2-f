# 系统接口标准

**版本**: 1.1.0  
**更新时间**: 2026-02-23  
**状态**: 正式版

---

## 1. 概述

### 1.1 接口定位

系统接口标准定义了**薯片内核**对外提供的所有标准API。内核使用 TypeScript 实现，运行在 Electron 主进程中。这些接口是生态中所有服务和插件与内核交互的唯一途径。主进程内的服务通过 `kernel.router.invoke()` 调用，插件（渲染进程）通过 Bridge API（`window.chips.invoke()`）调用。

### 1.2 接口分类

内核提供以下几类接口:

1. **卡片管理接口** - 卡片CRUD操作
2. **箱子管理接口** - 箱子CRUD操作
3. **资源管理接口** - 统一资源访问
4. **插件管理接口** - 插件生命周期管理
5. **路由管理接口** - 服务注册和路由
6. **事件系统接口** - 事件发布订阅
7. **文件系统接口** - 文件识别和操作
8. **配置管理接口** - 配置读写
9. **日志系统接口** - 日志记录
10. **认证授权接口** - 权限验证

### 1.3 调用方式

所有接口都通过薯片协议调用（插件侧统一使用 Bridge API）:

```javascript
// 渲染进程（插件）调用示例
const card = await window.chips.invoke("card", "read", {
  card_id: "a1B2c3D4e5"
});
```

> 迁移说明：旧的 `Core` 调用范式已归档，当前统一为 `window.chips.invoke(namespace, action, params)`。

## 2. 卡片管理接口

### 2.1 读取卡片 (card.read)

**服务名**: `card.read`  
**权限**: `card.read`

**输入参数**:
```json
{
  "card_id": "string (10位62进制ID)",
  "include_content": "boolean (可选,是否包含基础卡片配置,默认true)",
  "include_resources": "boolean (可选,是否包含资源列表,默认false)"
}
```

**输出数据**:
```json
{
  "card": {
    "id": "a1B2c3D4e5",
    "metadata": {
      "name": "卡片名称",
      "created_at": "2026-01-15T08:00:00.000Z",
      "modified_at": "2026-01-30T10:00:00.000Z",
      "theme": "薯片官方:默认主题",
      "tags": [...]
    },
    "structure": [
      {"id": "base1", "type": "VideoCard"},
      {"id": "base2", "type": "ImageCard"}
    ],
    "content": {
      "base1": {...},
      "base2": {...}
    },
    "resources": [...]
  }
}
```

### 2.2 创建卡片 (card.create)

**服务名**: `card.create`  
**权限**: `card.write`

**输入参数**:
```json
{
  "name": "string (卡片名称)",
  "theme": "string (可选,主题包)",
  "tags": "array (可选,标签数组)",
  "save_to": "string (保存路径,可选)"
}
```

**输出数据**:
```json
{
  "card_id": "new1Card2Id",
  "file_path": "/path/to/card.card"
}
```

### 2.3 更新卡片 (card.update)

**服务名**: `card.update`  
**权限**: `card.write`

**输入参数**:
```json
{
  "card_id": "string",
  "updates": {
    "metadata": {
      "name": "新名称",
      "tags": [...]
    },
    "structure": [...],
    "content": {...}
  }
}
```

**输出数据**:
```json
{
  "success": true,
  "modified_at": "2026-01-30T10:30:00.000Z"
}
```

### 2.4 删除卡片 (card.delete)

**服务名**: `card.delete`  
**权限**: `card.delete`

**输入参数**:
```json
{
  "card_id": "string",
  "permanent": "boolean (可选,是否永久删除,默认移到回收站)"
}
```

**输出数据**:
```json
{
  "success": true
}
```

### 2.5 添加基础卡片 (card.add_base_card)

**服务名**: `card.add_base_card`  
**权限**: `card.write`

**输入参数**:
```json
{
  "card_id": "string (复合卡片ID)",
  "base_card_type": "string (基础卡片类型)",
  "config": "object (基础卡片配置)",
  "position": "integer (可选,插入位置,默认末尾)"
}
```

**输出数据**:
```json
{
  "base_card_id": "newBase123",
  "success": true
}
```

### 2.6 移除基础卡片 (card.remove_base_card)

**服务名**: `card.remove_base_card`  
**权限**: `card.write`

**输入参数**:
```json
{
  "card_id": "string (复合卡片ID)",
  "base_card_id": "string (基础卡片ID)"
}
```

### 2.7 导出卡片 (card.export)

**服务名**: `card.export`  
**权限**: `card.read`

**输入参数**:
```json
{
  "card_id": "string",
  "export_path": "string",
  "mode": "string (full/empty,全填充或空壳)",
  "include_nested": "boolean (是否包含嵌套卡片)"
```

**服务名**: `resource.fetch`  
**权限**: `resource.read`

**输入参数**:
```json
{
  "uri": "string (chips://协议URI)",

    "auth_profile": "string (可选,认证配置名称)"

```

**输出数据**:
```json
{
  "uri": "chips://local/cards/a1B2c3D4e5/video.mp4",
  "content_type": "video/mp4",
  "size": 52428800,
  "data": "...(数据或数据流)",
  "metadata": {
    "cached": false,
    "source": "local",
    "last_modified": "2026-01-15T10:00:00.000Z"

}
```

**权限**: `resource.write`

**输入参数**:
```json

    "overwrite": "boolean (是否覆盖,默认false)",

}
```

### 4.3 检查资源存在 (resource.exists)

**权限**: `resource.read`

**输入参数**:
```json
{
  "uri": "string"
}
```

**输出数据**:
```json
{
  "exists": true,
```

### 4.4 获取资源元信息 (resource.meta)

**服务名**: `resource.meta`  
**权限**: `resource.read`

**输入参数**:
```json
{
  "uri": "string"
}
```

**输出数据**:
```json

  "modified_at": "...",
```

### 4.5 删除资源 (resource.delete)

**服务名**: `resource.delete`  
**权限**: `resource.delete`

**输入参数**:
```json

  "uri": "string",
```

### 12.3 查询标签 (tag.query)

**服务名**: `tag.query`  
**权限**: `tag.read`

**输入参数**:
```json
{
  "tags": [["地点", "日本"], "旅行"],
  "target_type": "string (可选)",
  "operator": "string (and/or,默认and)"
}
```

**输出数据**:
```json
{
  "results": [
    {
      "type": "card",
      "id": "a1B2c3D4e5",
      "name": "日本旅行",
      "matched_tags": [...]
    }
  ],
  "total": 25
}
```

### 12.4 标签建议 (tag.suggest)

**服务名**: `tag.suggest`  
**权限**: `tag.read`

**输入参数**:
```json
{
  "context": {
    "filename": "string (可选)",
    "content": "string (可选)",
    "existing_tags": ["tag1"]
  },
  "limit": "integer (默认10)"
}
```

**输出数据**:
```json
{
  "suggestions": [
    {"tag": "旅行", "confidence": 0.95},
    {"tag": ["地点", "日本"], "confidence": 0.88}
  ]
}
```
