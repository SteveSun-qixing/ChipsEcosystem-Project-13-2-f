# 数据模型设计

**版本**: 1.0.0  
**更新时间**: 2026-01-30  
**状态**: 正式版

---

## 1. 概述

本文档定义了Chips生态中所有核心数据结构的详细设计,包括:
- 基础卡片数据模型
- 箱子布局数据模型
- 用户数据模型
- 系统数据模型

所有数据模型都使用JSON Schema定义,确保类型安全和数据验证。

## 2. 基础卡片数据模型

### 2.1 通用字段

所有基础卡片配置都包含以下通用字段:

```yaml
$schema: "http://json-schema.org/draft-07/schema#"
title: "Base Card Common Fields"
type: object
required:
  - card_type

properties:
  card_type:
    type: string
    description: "基础卡片类型标识"
    
  theme:
    type: string
    description: "该卡片专属主题,为空则使用上级主题"
    default: ""
    
  layout:
    type: object
    properties:
      height_mode:
        type: string
        enum: ["auto", "fixed"]
        default: "auto"
      fixed_height:
        type: integer
        description: "固定高度(像素)"
      aspect_ratio:
        type: string
        pattern: "^\\d+:\\d+$"
        description: "宽高比,如 16:9"
```
