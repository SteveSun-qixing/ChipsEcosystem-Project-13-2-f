# 组件库使用指南

`@chips/components` 是官方无样式组件库，基于 Zag.js 状态机和 Ark UI 构建，提供完整的交互逻辑和无障碍支持。

##  设计理念

薯片组件库遵循"零样式组件"原则：

- 组件只提供结构和交互逻辑
- 所有视觉样式由主题包注入
- 使用 CSS 类名约定（`chips-*`）
- 完整的无障碍（a11y）支持

##  安装

```bash
pnpm add @chips/components
```

##  基础组件

### Button 按钮

```vue
<template>
  <ChipsButton
    variant="primary"
    size="md"
    :loading="isLoading"
    :disabled="isDisabled"
    @click="handleClick"
  >
    {{ t('common.save') }}
  </ChipsButton>
</template>

<script setup lang="ts">
import { ChipsButton } from '@chips/components';
</script>
```

变体（variant）：
- `primary` - 主要按钮
- `secondary` - 次要按钮
- `ghost` - 幽灵按钮
- `danger` - 危险按钮

尺寸（size）：
- `sm` - 小尺寸
- `md` - 中等尺寸（默认）
- `lg` - 大尺寸

##  CSS 类名约定

所有组件使用统一的 CSS 类名前缀 `chips-`：

```css
/* 组件根元素 */
.chips-button { }
.chips-input { }
.chips-select { }

/* 变体修饰符 */
.chips-button--primary { }
.chips-button--secondary { }
.chips-button--ghost { }

/* 尺寸修饰符 */
.chips-button--sm { }
.chips-button--md { }
.chips-button--lg { }

/* 状态修饰符 */
.chips-button--disabled { }
.chips-button--loading { }
.chips-input--invalid { }
.chips-input--focused { }

/* 子元素 */
.chips-button__icon { }
.chips-button__label { }
.chips-input__prefix { }
.chips-input__suffix { }
```

##  无障碍支持

所有组件内置完整的无障碍支持：

- 正确的 ARIA 属性
- 键盘导航支持
- 焦点管理
- 屏幕阅读器友好

```vue
<!-- 组件自动处理 ARIA 属性 -->
<ChipsButton>
  {{ t('common.save') }}
</ChipsButton>

<!-- 渲染结果 -->
<button
  class="chips-button chips-button--primary"
  role="button"
  aria-disabled="false"
>
  保存
</button>
```
