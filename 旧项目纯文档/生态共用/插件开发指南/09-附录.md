# 附录

### A. 常用 CLI 命令

| 命令 | 说明 |
|-----|------|
| `chipsd create <name>` | 创建新插件项目 |
| `chipsd dev` | 启动开发服务器 |
| `chipsd build` | 构建生产版本 |
| `chipsd pack` | 打包为 .cpk 文件 |
| `chipsd validate <file>` | 验证插件包 |
| `chipsd install <file>` | 安装本地插件 |
| `chipsd uninstall <id>` | 卸载插件 |
| `chipsd list` | 列出已安装插件 |
| `chipsd login` | 登录开发者账号 |
| `chipsd publish <file>` | 发布插件 |

### B. 错误码参考

| 错误码 | 说明 | 处理建议 |
|-------|------|---------|
| `PERMISSION_DENIED` | 权限不足 | 检查 manifest.yaml 权限声明 |
| `VALIDATION_ERROR` | 参数校验失败 | 检查传入参数格式 |
| `ROUTE_NOT_FOUND` | 服务或动作不存在 | 检查 namespace 和 action 名称 |
| `TIMEOUT` | 请求超时 | 增加超时时间或优化操作 |
| `USER_CANCELLED` | 用户取消操作 | 正常流程，无需处理 |
| `PLUGIN_NOT_FOUND` | 插件不存在 | 检查插件 ID 是否正确 |
| `FILE_NOT_FOUND` | 文件不存在 | 检查文件路径 |
| `PAYLOAD_TOO_LARGE` | 数据过大 | 使用资源 URL 替代直接传输 |
