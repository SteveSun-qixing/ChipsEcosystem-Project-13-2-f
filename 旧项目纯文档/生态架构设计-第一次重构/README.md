## 固定口径摘录
- 分层：12层（L1-L12）。
- 主题接口：`theme.list/theme.apply/getCurrent/getAllCss/resolve/contract.get`。
- 事件命名：点语义（`theme.changed`、`language.changed`）。
- Bridge 调用链：`UI Hooks -> Runtime Client -> Bridge Transport`。
- 布局契约：`owner: page`、`unit: cpx`、`baseWidth: 1024`。
- 模板契约边界：页面型插件强制 `ui.layout`，`theme` 插件豁免。
- 组件库：`@chips/component-library`。
