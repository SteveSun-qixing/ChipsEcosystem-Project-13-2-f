# 系统接口标准

## 概述

系统接口标准定义插件与操作系统之间的交互规范。这些接口封装系统能力，通过Bridge API向上层插件提供一致的访问方式。

## 平台检测接口

系统提供平台检测接口获取当前运行环境信息。platform.get()返回当前操作系统名称，如win32、darwin、linux。arch.get()返回系统架构，如x64、arm64。version.get()返回系统版本号。locale.get()返回系统语言设置。

## 窗口管理接口

窗口管理接口控制应用程序窗口。window.create(options)创建新窗口，options包含宽高位置等参数。window.close(windowId)关闭指定窗口。window.minimize(windowId)最小化窗口。window.maximize(windowId)最大化窗口。window.restore(windowId)恢复窗口。window.setTitle(windowId, title)设置窗口标题。window.setSize(windowId, width, height)设置窗口尺寸。window.setPosition(windowId, x, y)设置窗口位置。window.setAlwaysOnTop(windowId, flag)设置窗口置顶。window.setFullScreen(windowId, flag)设置全屏模式。

## 屏幕管理接口

屏幕管理接口获取显示器信息。screen.getAll()返回所有显示器列表，每个显示器包含id、bounds、workArea等信息。screen.getPrimary()返回主显示器信息。screen.getCurrent()返回包含指定窗口的显示器。screen.onDisplayChanged监听显示器变化事件。

## 系统对话框接口

系统对话框接口调用原生对话框。dialog.showOpenDialog(options)显示打开文件对话框，options包含title、defaultPath、filters、properties等。dialog.showSaveDialog(options)显示保存文件对话框。dialog.showMessageBox(options)显示消息对话框。dialog.showErrorBox(title, content)显示错误对话框。dialog.showItemInFolder(path)在文件管理器中显示文件。

## 剪贴板接口

剪贴板接口访问系统剪贴板。clipboard.readText()读取剪贴板文本。clipboard.writeText(text)写入文本到剪贴板。clipboard.readHTML()读取HTML格式内容。clipboard.writeHTML(html)写入HTML内容。clipboard.readImage()读取图片内容。clipboard.writeImage(image)写入图片到剪贴板。clipboard.clear()清空剪贴板。

## 系统通知接口

系统通知接口显示系统通知。notification.show(options)显示通知，options包含title、body、icon、silent等。notification.onClick(callback)监听通知点击事件。notification监听通知关闭事件.onClose(callback)。

## 快捷键接口

快捷键接口注册全局快捷键。globalShortcut.register(accelerator, callback)注册全局快捷键，accelerator是快捷键字符串如CommandOrControl+Shift+N。globalShortcut.unregister(accelerator)注销快捷键。globalShortcut.unregisterAll()注销所有已注册的快捷键。globalShortcut.isRegistered(accelerator)检查快捷键是否已注册。

## 电源管理接口

电源管理接口控制电源行为。powerMonitor.onSuspend(callback)监听系统挂起事件。powerMonitor.onResume(callback)监听系统恢复事件。powerMonitor.onACPowerChange(callback)监听电源连接变化。powerMonitor.onBatteryPowerChange(callback)监听电池电量变化。

## 应用菜单接口

应用菜单接口配置应用程序菜单。Menu.setApplicationMenu(menu)设置应用菜单，menu是菜单结构对象。Menu.getMenuItem(id)获取菜单项。Menu Popup.contextMenu(template)在指定位置显示上下文菜单。

## 文件关联接口

文件关联接口处理文件关联。setAsDefaultProtocolClient(protocol)将应用注册为协议处理器。removeAsDefaultProtocolClient(protocol)移除协议处理器。isDefaultProtocolClient(protocol)检查是否为默认协议处理器。

## 系统托盘接口

系统托盘接口管理系统托盘图标。tray.create(options)创建托盘图标，options包含icon、tooltip、menu等。tray.setImage(image)设置图标。tray.setToolTip(tooltip)设置提示文本。tray.setContextMenu(menu)设置右键菜单。tray.destroy()销毁托盘图标。

## 电源状态接口

电源状态接口获取电源信息。getPowerMonitor()返回电源监控对象。getSystemIdleTime()返回系统空闲时间。setPreventSleep(prevent)设置是否阻止系统休眠。getTotalMemory()返回系统总内存。getFreeMemory()返回可用内存。

## 打印机接口

打印机接口获取打印机列表和打印功能。printers.getAll()返回所有打印机列表。printers.getDefault()返回默认打印机。window.print(options)打印当前窗口内容。

## 兼容性处理

不同操作系统对系统API的支持存在差异。系统接口提供统一的抽象，底层自动适配平台差异。某些API在特定平台不可用时，接口返回适当的错误或替代实现。开发者在使用系统接口时应处理可能的平台差异。

## 服务域总表（vNext 冻结口径）

系统接口标准覆盖以下服务域：

| 服务域 | 核心动作 | 说明 |
|---|---|---|
| `platform` | `getInfo/getCapabilities/openExternal/dialog*/clipboard*/shell*/notificationShow/tray*/shortcut*/power*` | 平台系统能力聚合 |
| `window` | `open/focus/resize/setState` | 窗口管理 |
| `file` | `read/write/stat/list` | 文件操作 |
| `theme` | `list/apply/getCurrent/getAllCss/resolve/contract.get` | 主题服务 |
| `i18n` | `getCurrent/setCurrent/translate/listLocales` | 国际化服务 |
| `config` | `get/set/batchSet/reset` | 配置管理 |
| `log` | `write/query/export` | 日志服务 |
| `credential` | `get/set/delete/rotate` | 凭证管理 |
| `plugin` | `install/enable/disable/uninstall/query` | 插件管理 |
| `module` | `mount/unmount/query/list` | 模块管理 |
| `card` | `parse/render/validate` | 卡片处理 |
| `box` | `pack/unpack/inspect` | 箱子处理 |
| `zip` | `compress/extract/list` | ZIP 处理 |
| `serializer` | `encode/decode/validate` | 序列化 |
| `control-plane` | `health/check/metrics/diagnose` | 治理聚合层 |

说明：`window.chips.dialog/*`、`window.chips.clipboard/*`、`window.chips.shell/*` 作为 Bridge 子域继续保留，但在主机路由层统一映射为 `platform.*` 动作，避免服务域口径分叉。

## 主题服务接口

theme 服务提供主题管理能力（vNext 冻结口径）：

| 动作 | 说明 | 幂等 |
|---|---|---|
| `theme.list({ publisher? })` | 获取所有已安装主题列表 | 是 |
| `theme.apply({ id })` | 应用指定主题 | 否 |
| `theme.getCurrent({ appId?, pluginId? })` | 获取当前生效主题 | 是 |
| `theme.getAllCss()` | 获取当前主题完整 CSS | 是 |
| `theme.resolve({ chain })` | 解析主题作用域链 | 是 |
| `theme.contract.get({ component? })` | 获取主题契约接口点 | 是 |

## 国际化服务接口

i18n 服务提供多语言能力（vNext 冻结口径）：

| 动作 | 说明 | 幂等 |
|---|---|---|
| `i18n.getCurrent()` | 获取当前语言环境 | 是 |
| `i18n.setCurrent({ locale })` | 设置当前语言环境 | 否 |
| `i18n.translate({ key, params? })` | 翻译指定键 | 是 |
| `i18n.listLocales()` | 获取已安装语言列表 | 是 |

## 配置服务接口

config 服务提供应用配置管理能力：

| 动作 | 说明 | 幂等 |
|---|---|---|
| `config.get({ key })` | 读取配置项 | 是 |
| `config.set({ key, value })` | 设置配置项 | 否 |
| `config.batchSet({ entries })` | 批量设置 | 否 |
| `config.reset({ key? })` | 重置配置 | 否 |

## 日志服务接口

log 服务提供日志记录与查询能力：

| 动作 | 说明 | 幂等 |
|---|---|---|
| `log.write({ level, message, data? })` | 写入日志 | 否 |
| `log.query({ filter, limit? })` | 查询日志 | 是 |
| `log.export({ format?, dateRange? })` | 导出日志 | 是 |

## 凭证服务接口

credential 服务提供敏感凭证管理能力（高敏感，需独立审计日志）：

| 动作 | 说明 | 幂等 |
|---|---|---|
| `credential.get({ ref })` | 获取凭证 | 是 |
| `credential.set({ ref, value })` | 设置凭证 | 否 |
| `credential.delete({ ref })` | 删除凭证 | 否 |
| `credential.rotate({ ref })` | 轮转凭证 | 否 |

## 控制平面服务接口

control-plane 是治理聚合层，提供系统级监控与诊断能力：

| 动作 | 说明 | 幂等 |
|---|---|---|
| `control-plane.health()` | 健康检查 | 是 |
| `control-plane.check({ scope })` | 状态检查 | 是 |
| `control-plane.metrics({ types? })` | 获取指标 | 是 |
| `control-plane.diagnose({ scope })` | 诊断报告 | 是 |
