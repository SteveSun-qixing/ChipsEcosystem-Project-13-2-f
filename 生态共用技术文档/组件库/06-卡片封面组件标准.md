# 卡片封面组件标准

## 1. 组件定义

`CardCoverFrame` 是生态统一卡片封面显示组件，输出为 iframe 窗口 + 下方卡片名称。

## 2. 使用场景

- 资源管理器卡片列表
- 卡片选择器
- 导入预览
- 编辑引擎侧边封面列表

## 3. 功能要求

- iframe 显示卡片封面 HTML
- 下方显示卡片名称
- 加载、空态、失败态完整
- 对外事件：打开卡片、加载完成、加载失败

## 4. 契约要求

- `data-scope="card-cover-frame"`
- `data-part="root|frame-container|iframe|name|status"`
- `data-state="idle|loading|ready|error|disabled|empty"`

## 5. 安全要求

- iframe 必须使用受限 sandbox
- sandbox 默认策略至少包含 `allow-scripts allow-same-origin`，并禁止危险权限（top-navigation、popups-to-escape-sandbox）
- 封面资源 URL 必须来自受控 SDK 接口
- 禁止任意注入 HTML 字符串

## 6. 验收要求

- 封面可稳定加载
- 卡片名称显示正确
- 失败可回退并给出标准错误
- 主题切换与语言切换不破坏布局
