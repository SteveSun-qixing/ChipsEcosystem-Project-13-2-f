# 工单005-ECOSYS-STAGE10-INTEGRATION-ACCEPTANCE 任务说明

## 1. 背景

阶段十要求组件库在三类典型插件场景完成生态接入验收：

- 设置面板插件
- 卡片编辑器插件
- 卡片查看器插件

当前仓库缺少可执行的三类插件验收工程，同时卡片显示链路仍受 SDK 工单阻断，导致端到端验收无法闭环。

## 2. 阻断项

1. 三类插件验收工程缺失（无法执行实机接入回归）。
2. `工单001-SDK-UNIFIED-CARD-DISPLAY-API` 未闭环（编辑器/查看器卡片显示链路不可联调）。

## 3. 协作交付物

### 3.1 生态集成验收组

- 提供统一联调沙箱或独立验收工程（settings/editor/viewer）。
- 提供标准卡片数据集与回归场景脚本。

### 3.2 SDK 开发组

- 闭环统一显示接口工单：`工单001-SDK-UNIFIED-CARD-DISPLAY-API`。
- 交付可联调 SDK 版本与契约测试记录。

### 3.3 编辑器与查看器项目组

- 接入组件库最新发布版本。
- 按模板完成主题切换、语言切换、配置变更、卡片显示链路回归。

## 4. 组件库项目组待执行任务（协作完成后）

1. 基于联调工程执行三类场景正式验收。
2. 更新 `reports/ecosystem/ecosystem-acceptance-template.json` 为正式验收报告。
3. 运行 `npm run validate:ecosystem-acceptance` 输出结构校验报告。
4. 回填阶段十计划文档与阶段十退出条件状态。

## 5. 验收标准

- 三类场景全部状态为 `passed`。
- 验收报告结构校验通过。
- 阶段十计划从“进行中”更新为“已完成”。
- 相关阻断工单状态更新为“已闭环”。
