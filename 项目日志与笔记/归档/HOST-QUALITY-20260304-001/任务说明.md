# HOST-QUALITY-20260304-001 任务说明

## 1. 任务背景

本任务对应工单 `HOST-QUALITY-20260304-001`，用于闭环逐文件审查后确认的高优质量问题，目标是将 Chips Host 提升到可验收的高质量交付状态。

## 2. 问题分解

- 修复 `prefix` 匹配误命中问题。
- 强化 EventBus 的过滤与超时订阅能力。
- 维持幂等、防重放、熔断机制的稳定性与兼容性。

- 补齐 `useWindow/usePlugin/useConfig` 等标准 Hooks。

### 2.3 服务域与契约

- 强化 schema 校验粒度，避免 response 全量放行。

- 保持 16 服务动作面与契约一致。

### 2.4 Plugin Runtime 与安全

- 增强 manifest 准入校验与安装期安全检查。
- 补充会话与运行态审计信息。
- 完善失败路径与可观测性。

### 2.5 PAL 与 Bridge

- 补齐 PAL 屏幕信息能力。
- 落地 Bridge IPC 子能力通道（兼容主通道）。
- 保持 contextBridge 与最小权限模型不回退。

## 3. 拟定交付清单
