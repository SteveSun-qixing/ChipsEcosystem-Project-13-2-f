# 工单004-ECOSYS-TOOLCHAIN-TS-LINT-SD-PRESET 任务说明

## 1. 任务背景

薯片组件库阶段一已完成架构与契约冻结；阶段二、阶段三已完成本地工程化与 token 流水线基础能力，并通过 `npm run verify`。

当前阻断点不在组件库代码质量本身，而在“生态统一工具链口径”尚未以可复用工程产物形式交付，导致阶段二/三收口无法按生态标准宣布完成。

## 2. 阻断范围

### 2.1 阶段二阻断

- 需接入生态统一 TypeScript strict 门禁 preset。
- 需接入生态统一 ESLint + Prettier preset（含规则白名单和禁用项）。
- 需将上述门禁稳定并入组件库聚合校验入口。

### 2.2 阶段三阻断

- 需接入生态统一 Style Dictionary 构建 preset。
- 需使用生态统一 token schema 校验器与 transform 规则。
- 需确保产物命名、目录、格式与主题系统完全一致。

## 3. 协作交付物要求

  - 输出模板规范
  - schema 校验插件
- 交付 token 产物口径文档（包含 diff 报告字段约定）。

## 4. 组件库项目组补充任务（待协作方交付后执行）
